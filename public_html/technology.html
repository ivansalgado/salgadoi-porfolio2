Technology
HTML & CSS

Our web sites at the App Academy are developed with a computer application called NetBeans. This application enables us to use HTML (Hypertext Markup Language). This is a system that you can use for creating a web site. As shown below, web sites are composed of a large variety of elements. Each element is enclosed in angle brackets and comes in a pair. For example, "< article >" and "< /article >" would be in charge of displaying any text on a web page. CSS is another language use in computer programming. CSS stands for Cascading Style Sheet. It's used for designing the look of a web site. For example, to make the color of the font white, you would type in: body{color: white;}.

Bootstrap

Bootstrap is an open-source software, used to assist HTML and CSS. This means that you can add certain features (such as the dropdowns used in the index page) to make your web site look sleeker and more professional.
