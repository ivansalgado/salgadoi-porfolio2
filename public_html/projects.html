Snake

Snake is a game that was originally created in the 1970s. The object of the game is to increase the length of your winding snake by eating "food". However, you must not run into any walls or your snake! Use any of the arrow keys to move the snake.

Unlike the previous projects, Snake is created by a computer program called JavaScript (JS). JS is a more complex, more fragile program than HTML or CSS. Therefore, one minor flaw in the code can cause the game to malfunction.

Snake What is JavaScript?
Mario

Der Mario, based on Nintendo's Super Mario, is another game that consists of mainly HTML and JavaScript. Like Snake, you can use the arrow keys to direct the player through each course.

For this project, two new programs are introduced: Tiled and melonJS, both of which assist the structure of the game. Tiled is used to draw the map while melonJS assists in the actual coding of the game.
